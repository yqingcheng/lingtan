环境：

- nodejs 版本 v18.16.0
- npm 版本 9.5.1
- hexo 版本 6.3.0
- hexo主题：butterfly 4.9.0

---



下载源代码：

```bash
git clone git@github.com:StarDreamAndFeng/StarDreamAndFeng.github.io.git
```

安装依赖：

```bash
npm install
```

运行：

```bash
npm run server
```
