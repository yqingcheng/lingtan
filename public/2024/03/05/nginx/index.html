三层架构

三层结构：（具体以下图为例）

  • 全局块
  • event 块
  • http 块
    • http 全局块
    • 多个 server 块
      • server 全局块

Listen

server块中最重要的指令就是listen指令，这个指令有三种配置语法。这个指令默认的配置值是：listen *:80 | *:8000；只能在server块种配置这个指令。

  1. 监听某个ip地址
  2. 监听某个端口（建议 ）
  3. 其他（无需多加关注）

拓展：可以简单了解server_name，它可以准确匹配、通配符（前后）以及正则匹配。
